## 其他

- 字符串（stringx.lua）
  - split：字符分割
- 表（tablex.lua）
  - copy：深拷贝
- 数组（arrayx.lua）
  - min：最小值
  - max：最大值
  - sum：总计
  - any：存在任意满足条件
  - filter：过滤
  - find：查找
  - find_index：查找索引
  - merge：合并数组
  - for_each：遍历
  - map：处理
  - reduce：消减
  - reverse：反转
